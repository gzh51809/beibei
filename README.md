### 贝贝网

母婴app,基于react开发。

浏览地址 ：  [beibei]( http://39.105.231.83:2996)。

运行：npm start，端口： localhost:3000。 



#### 项目工具

| 选项               | 工具                        |
| ------------------ | --------------------------- |
| 开发环境           | NodeJS                      |
| UI框架             | Ant Design Mobile，iconfont |
| JavaScript的框架   | React.js                    |
| 开发工具           | VSCode                      |
| 前端构建工具       | WebPack                     |
| 包管理工具         | NPM                         |
| 打包的应用程序工具 | DCloud                      |



#### 目录

```
\根目录
├── README.md					项目说明文档
├── package.json				Node.js模块描述文件
├── package-lock.json			模块版本锁定文件
├── .gitignore					忽略文件目录
├── config/							配置文件
├── public/						公共文件
├── scripts/					npm scripts
└── src/						源文件
```
